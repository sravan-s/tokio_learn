## Simple Broadcast Channel

- Server `cargo run --bin server`
- Client
- Auth

### How to run?

* Run server with `cargo run --bin server`
* Run client with `cargo run --bin client`
    * or open telnet connection `telnet localhost 8080`
* Try sending messages ~
